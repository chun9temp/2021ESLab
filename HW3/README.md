# HW3 #

Send sensor data via socket and visualize sensor data

## Usage ##

1. Edit ```mbed_app.json``` to include the correct SSID and password.

2. Set ```SocketAddress addr("IP",port)``` in ```main.cpp```.

3. Set ```HOST``` and ```PORT``` in ```server.py```.

4. Run ```server.py```.

5. Run HW3 on STM32.
